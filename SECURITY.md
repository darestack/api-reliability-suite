# Security Policy

## Supported Versions

This repository is a template and does not publish versioned releases. Security fixes are applied to the `main` branch. If you fork or deploy this code, you are responsible for updating your deployment regularly.

## Reporting a Vulnerability

If you find a security issue, please report it privately instead of opening a public GitHub issue.

- Email: adelekedare2012@gmail.com

Please include:
- A clear description of the issue
- Steps to reproduce
- Impact assessment (if known)
- Any suggested fixes or mitigations

We will acknowledge receipt and respond with a remediation plan as quickly as possible.
